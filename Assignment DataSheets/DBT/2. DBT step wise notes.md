**dbt = SQL + Version Control + Testing + Documentation + Deployment**

dbt is a framework used by Data Engineers & Analytics Engineers to **transform data inside the data warehouse or lakehouse**.

dbt does NOT extract or load data.
It only does the **T (Transform) in ELT**.
