
### Step 1 – Create a new venv with Python 3.8

In your project folder (`C:\Personal\Revature\Day 8\Air Flow\Demos`):

```bash
py -3.8 -m venv .venv38
```

This creates a virtual env named `.venv38` using Python 3.8.

---

### Step 2 – Activate the venv

```bash
.\.venv38\Scripts\activate
python --version
```

You should now see something like:

```text
Python 3.8.x
```

If yes, you’re in the right environment.

---

### Step 3 – Install Airflow 2.9.0 with the correct constraints

Now run one single line:

```bash
pip install "apache-airflow==2.9.0" --constraint "https://raw.githubusercontent.com/apache/airflow/constraints-2.9.0/constraints-3.8.txt"
```
